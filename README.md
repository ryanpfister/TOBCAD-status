# TOBCAD Status

Public status page for TOBCAD bugs, feature requests, and suggestions.

## How to update
Edit `status.json` and commit changes.

## Item format
```json
{
  "title": "Issue title",
  "type": "bug",
  "status": "Open",
  "updated": "2026-04-22",
  "description": "Short public-friendly description.",
  "link": "https://github.com/ryanpfister/TOBCAD/issues"
}
